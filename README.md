Desafio proposto na formação Github Certification da dio.me

1. Criar uma conta no Github:
2. Criar um repositório: 
   + [x] Crie um novo repositório no GitHub, nomeando-o como "desafio-github-markdown 
   + [x] Adicione o seu README.md ao criar o repositório
   + [x] Utilize o Github.dev para editar seu projeto diretamente com o editor web do GitHub.
   + [x] Deixar o repositório público

3. Colaboração:
   + [ ] Adicione um colaborador ao seu repositório (pode ser um colega de curso ou você mesmo com uma conta diferente)
   + [ ] Ambos devem fazer um clone do repositório em suas máquinas locais
   + [ ] Realizar alterações, commits, abrir issues e pull requests

4. Formatação com Markdown:
   + [x] Edite o arquivo "README.md" em seu repositório
   + [x] Utilize a linguagem markdwon para formatar informações sobre o desafio, instruções e uma breve descrição do seu projeto.
